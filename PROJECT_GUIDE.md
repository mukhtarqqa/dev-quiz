# 📚 Документация проекта DevQuiz

Этот проект представляет собой современную платформу для IT-тестирования.

---

## 🏗️ Структура проекта
*   `src/App.jsx` — Логика приложения, экраны и управление состоянием.
*   `src/index.css` — Дизайн-система и адаптивные стили.
*   `src/questions.js` — База данных статических тестов.

---

## 🎨 Дизайн-система (CSS Variables)

| Переменная | Описание |
| :--- | :--- |
| `--bg` | Основной фон. |
| `--accent` | Цвет акцента (меняется темами). |
| `--glass-bg` | Фон для эффекта «стекла». |

---

## 🧩 Навбар (Adaptive Header)

### 🖥️ Десктоп (Desktop)
*   **Островной режим**: По умолчанию парит сверху.
*   **Режим скролла**: Расширяется на **всю ширину окна** и становится прямоугольным.

### 📱 Мобильные устройства (Mobile < 620px)
*   **Всегда прямоугольный**: Островной режим отключен.
*   **Без выбора языка**: Селектор языка скрыт (доступен в профиле).

---

## 🛠️ Частые правки

### Изменение ширины контента
```css
#root { max-width: 620px; }
```

### Добавление новых админов
В `App.jsx` добавь email в массив:
```javascript
const ADMIN_EMAILS = ['...', 'new-admin@gmail.com'];
```
